Kamis, 26 Maret 2015

Pemerintahan Wajib Sosialisasikan BPJS Ketenagakerjaan Secara Masif

JAKARTA (26 Maret): Anggota Komisi IX DPR dari Fraksi NasDem Irma Suryani Chaniago mengingatkan bahwa permerintahan wajib mensosialisasikan program BPJS Ketenagakerjaan secara Komprehensif, masif dan sistematis.

"Jangan sampai pengusaha membohongi lagi para pekerjanya" tegas Irma dalam diskusi BPJS Ketenagakerjaan yang digelar Gerakan Massa Buruh (Gemuruh) Partai NasDem di Jakarta, Kamis (26/3).
